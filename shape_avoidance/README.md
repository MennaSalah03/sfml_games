# Avoidant Circle

This is a game about a little circle that is avoidant. It fears and avoids everything that isn't familiar, but by time it realized that sometimes it's worth it to interact with the world without avoiding everythin and letting fear control the game

The color pallete:
I used the *Pixel Pastels* from a blog post on [media.io](https://www.media.io/color-palette/gaming-color-palette.html). <br>
colors: HEX: #FFD6E8 #BDE0FE #CDB4DB #FFF1B6 #2B2D42


## Current Mechanism
The game is controlled by a **mouse** for motion, and you can control the size of the circle using **left** and **right** clicks for increasing and deceasing the player size respectively.
