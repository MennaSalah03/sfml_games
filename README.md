# SFML Games

This repository is for creating small/simple games in C++ SFML library. I am currently doing project-based learning while getting guidance from this [playlist](https://www.youtube.com/playlist?list=PL6xSOsbVA1eb_QqMTTcql_3PdOiE928up).

This project is for fun purposes as a hobby to wind down from other heavier projects and the passion to make more complicated cozy games in the future.
